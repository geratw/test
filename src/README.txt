Smart Calculator
О программе
Калькулятор реализован на графической библиотеке QT.Есть возможность вызвать 3 вида калькулятора(Обычный, кредитный и чуть позже будет доступен депозитный) также есть возможность построения графика, для этого введите график функции, укажите диапазон отображаемой области определения и область значения и нажмите кнопку график
Чтобы начать пользоваться калькулятором, его необходимо установить с помощью команды make install. Эта команда установит программу в папку build
Чтобы удалить программу, нужно воспользоваться командой uninstall.
